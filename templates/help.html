Help and tutorials

Guidance is organized around the same workflow used by the server: upload, pre-correction assessment, correction, post-correction assessment, and export.

Upload data

Upload a count matrix and metadata table in CSV format, then map the required metadata fields. The example anaerobic digestion dataset remains available for a quick walkthrough.

Pre-correction assessment

Inspect study design, ordination structure, batch association, and phenotype separation before correction so technical variation can be interpreted in context.

Batch effect correction

Run compatible correction methods with curated defaults or adjust method-specific parameters when the study design requires it.

Post-correction assessment

Compare before and after diagnostics, review correction trade-offs, and download result bundles for downstream analysis.
